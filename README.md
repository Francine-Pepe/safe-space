## SAFE SPACE

Safe Space is a calm, distraction-free web application designed to support neurodivergent users during moments of stress, overwhelm, or emotional overload.

Built with Next.js, the app provides simple and accessible tools that help users regulate their emotions without adding pressure, complexity, or cognitive load. Instead of focusing on productivity, Safe Space focuses on comfort, self-regulation, and creating a sense of safety.

Current features include:

* Guided breathing exercises with visual cues
* A calming decision flow for moments of overwhelm
* Accessibility-focused design with support for reduced motion preferences
* A clean, low-stimulation interface designed for mobile and desktop devices

Planned features include:

* Brain dump / journaling space
* Sensory regulation tools and ambient sounds
* Comfort box with personalized content
* Dark mode and low-stimulation themes
* Offline support through Progressive Web App (PWA) capabilities

This project was originally created as a personal tool for a neurodivergent teenager, with the goal of providing a safe digital space that feels supportive, predictable, and easy to use during difficult moments.
